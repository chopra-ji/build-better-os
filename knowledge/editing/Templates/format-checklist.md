## Purpose

This checklist is used at the `[stage]` stage of production for `[format]`. Complete it before handing off to the next stage or publishing.

## Pre-Submission (Author)

- [ ] Headline: does it earn the click without misleading the reader?
- [ ] Opening: does the first paragraph justify reading the second?
- [ ] Argument: is there one clear argument, or are there multiple competing ideas that need to be resolved?
- [ ] Evidence: are all factual claims sourced or sourceable?
- [ ] Length: is every sentence earning its place?
- [ ] Voice: is this consistent with Build Better's editorial voice?

## Copy Edit (Editor)

- [ ] Grammar and style: consistent with house style rules in `editing/INDEX.md`
- [ ] Facts: key factual claims verified against primary sources
- [ ] Links: all external links working and pointing to intended destinations
- [ ] Headlines and subheadings: parallel structure, appropriate register
- [ ] Pull quotes: if present, accurate transcription from the text
- [ ] Byline and metadata: correct

## Pre-Publication (Production)

- [ ] Images: correct dimensions, alt text present, copyright cleared
- [ ] SEO metadata: title tag, meta description, canonical URL
- [ ] Internal links: at least one link to a related piece where appropriate
- [ ] Legal: no uncleared trademarks, no defamatory claims, no copyright issues
- [ ] Scheduling: confirmed publish time matches editorial calendar

## Post-Publication (Within 24 Hours)

- [ ] Live check: article renders correctly on all target platforms
- [ ] Analytics: tracking is firing correctly
- [ ] Social: distributed to relevant channels
