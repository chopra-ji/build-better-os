# Resources

Curated external references for this domain.

## Purpose

Resources are external links, tools, publications, and references that have earned a permanent place in this domain's knowledge base. This is not a bookmarks folder — every resource here has a documented reason for being here and a documented use case.

## What Belongs Here

- Primary sources frequently referenced in entries (journals, official documentation, foundational papers)
- Tools actively used in this domain's workflows
- Newsletters, publications, or feeds that consistently produce high-signal content for this domain
- Reference materials that contributors should know about before researching in this domain

## What Does NOT Belong Here

- Links that are only relevant to one entry (those go in the entry's own frontmatter)
- General web searches that can be reproduced in seconds
- Resources that have not been evaluated for quality

## Resource Format

Each resource listed here should have a corresponding entry in Templates/ created with `resource.md`. The entry documents: what it is, why it is here, how to use it, and its limitations.

## Resource List

Resources will be added as the domain is built out. When adding a resource, create its entry file in the parent domain directory using the resource template, then add a reference row here.
