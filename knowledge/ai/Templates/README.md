# AI Domain Templates

Templates for entries in the AI knowledge domain. These extend the base templates in `knowledge/_templates/` with AI-specific fields.

| Template | Use For |
|---|---|
| [model-evaluation.md](model-evaluation.md) | Evaluating a specific AI model for Build Better use cases |
| [tool-review.md](tool-review.md) | Reviewing an AI-powered tool or product |
| [research-note.md](research-note.md) | Synthesizing AI research, papers, or capabilities analysis |
